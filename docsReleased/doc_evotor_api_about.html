    Об Облаке Эвотор

    Раздел содержит вводную информацию для работы с нашей документацией.

    Облако Эвотор позволяет интегрировать сторонний сервис и смарт-терминал. В Облаке хранятся следующие данные пользователей платформы:

    • Магазины;
    • Сотрудники;
    • Смарт-терминалы;
    • Товары;
    • Документы смарт-терминала.

    Для доступа к данным облака используйте Эвотор API. Эвотор API – это REST API, который поддерживает методы как для доступа к данным пользователя, так и для взаимодействия со сторонним сервисом. Все методы Эвотор API описаны в справочние API.

    Методы, которые дают доступ к данным описаны в разделах:

    • Сущности пользователя Эвотор;
    • Товары и документы;
    • Схемы и дополнительные поля товаров.

    Методы, собранные в разделах вебхуки-запросы и веб-хуки уведомления, позволяют стороннему сервису получать изменения, которые происходят в Облаке.

    Установленные на смарт-терминале приложения не работают с Облаком напрямую и получают данные с помощью сторонних сервисов.
